# EmoSafe AI

Simple Streamlit app that runs basic test prompts to observe how AI systems respond to emotionally sensitive or unclear inputs.

## What it does
- Runs a small set of test prompts
- Displays simulated evaluation output
- Placeholder for future AI model integration

## Why I built this
To explore how AI systems behave in emotionally sensitive contexts and identify basic failure patterns in responses.

## Status
Early prototype — backend integration coming next.
